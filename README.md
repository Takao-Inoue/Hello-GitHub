## Backslash escapes

\*literal asterisks\*   

\*\*\*\*\*\*\*\*\*\*  (ten asterisks)

Markdown(GFM) provides backslash escapes for the following characters:

* \\   backslash
* \`   backtick
* \*   asterisk
* \_   underscore
* \{\}  curly braces
* \[\]  square brackets
* \(\)  parentheses
* \#   hash mark
* \+   plus sign
* \-   minus sign (hyphen)
* \.   dot
* \!   exclamation mark
